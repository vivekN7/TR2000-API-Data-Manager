System Overview ERD

High-level overview showing the main entities and their relationships.
erDiagram OPERATORS ||--o{ PLANTS : operates PLANTS ||--o{ ISSUES : has PLANTS ||--o{ PCS : contains ISSUES ||--o{ REFERENCES : includes PCS ||--o{ PCS_DETAILS : has VDS ||--o{ VDS_SUBSEGMENTS : contains ETL_CONTROL ||--o{ ETL_LOGS : tracks

Key Entities

OPERATORS: operator_id (PK), operator_name

PLANTS: plant_id (PK), operator_id (FK), long_description

ISSUES: plant_id (FK), issue_revision (PK), status

PCS: plant_id (FK), pcs (PK), revision, rating_class

VDS: vds (PK), revision, description

ETL_CONTROL: etl_run_id (PK), run_date, status
